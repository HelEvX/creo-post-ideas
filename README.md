# Creo Post Ideas Tool

A Vue 3 **theme-switching MVP** developed for [Creo Websolutions](https://www.creowebsolutions.be/).  
This tool previews client-specific brand themes in a browser-based interface, allowing users to visualize and copy their color palettes, fonts, and style tokens for use in social media content.
